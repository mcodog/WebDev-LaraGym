# WebDev-Laragym

<h2>Start Up</h2>
Run the following commands

composer install
cp .env.example .env

php artisan key:generate
php artisan migrate
php artisan optimize

npm install
npm run dev

php artisan serve

*optional*
php artisan db:seed
